# E-commerce Analytics Dashboard

## Overview
End-to-end data analytics project analyzing e-commerce data using Python and Power BI.

## Tools Used
- Python (Pandas)
- Power BI

## Key Metrics
- GMV (Total Revenue)
- Total Orders
- Avg Delivery Time
- SLA Breach Rate

## Insights
- Revenue is growing steadily over time
- Delivery delays impact SLA performance
- Revenue is concentrated among top customers

## Dataset
Dataset used: Brazilian E-commerce Public Dataset by Olist  
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
